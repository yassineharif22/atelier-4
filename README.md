# Analyseur de Texte IA
Application Streamlit d'analyse statistique de texte.
## Installation locale
```bash
pip install -r requirements.txt
streamlit run app.py
```
## Variables d'environnement
| Variable | Description | Défaut |
|----------|-------------|--------|
| APP_TITLE | Titre affiché | Analyseur de Texte IA |
| APP_VERSION | Version affichée | 1.0.0 |
